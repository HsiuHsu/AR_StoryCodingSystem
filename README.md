# AR Story Coding System
## Introduction

AR Story Coding System 是一個虛實交互的互動系統，此專案目的為給予特殊教育孩童進行社交學習。

## Software Used

採用Unity進行開發，運用AR Foundation技術進行虛實的交互設計。
